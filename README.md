## Issues Faced and How I Solved Them

### Issue 1: Understanding GitHub workflow

I was initially new to the GitHub workflow, especially the difference between creating a repository, committing changes, and pushing changes.

**Solution:**

I understood that:
- A repository is the project folder stored on GitHub.
- A commit saves my changes locally.
- A push uploads those changes to GitHub.

### Issue 2: Finding and installing the correct extensions

I had to search carefully inside Cursor Extensions to find the Claude Code and Codex extensions.

**Solution:**

I verified the extension names and installed both extensions successfully from the Cursor Extensions section.
